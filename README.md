# GTA V Auto Splitter

A <a href="https://livesplit.org/" target="_blank">LiveSplit</a> Auto Splitter made for GTA V. This is a modified version of the `GTAVsplitter_hundo.asl` file by `burh√°c#5472`. This Auto Splitter is made for version 1.27 of GTA V, which is the version used by speedrunners.


For any questions, ask in the GTA V Speedrunning Discord: https://discord.gg/3qjGGBM

# Build
1. Get Rust from https://rustup.rs.
2. Add wasm32 target with `rustup target wasm32-unknown-unknown`.
3. Build with `cargo build --target wasm32-unknown-unknown --release`.
